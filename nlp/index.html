AI for Natural Language Processing

So natural language processing (NLP) is a sub-field of linguistics, computer science, information enginering, and artificial inteligence concerned with the interactions between computers and human (natural) languages, in particular how to program computers to process and analyze large amounts of natural language data.

Challenges in in natural language processing frequently involve speech recognition, natural language understanding, and natural language generation.

There are many people using these technologies today, while some think AI is a loose cannon, others work very hard to make improve it.
